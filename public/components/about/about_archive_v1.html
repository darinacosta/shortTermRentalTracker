About the Nola Short Term Rental Report

The Nola Short Term Rental Report is a living document. The content and data is likely to improve as research and analysis continue. The listings featured on the map are collected using the Zilyo API. After these listings are compiled, the individual listings in this compiled list are analyzed in order to collect additional data on the listings themselves and on user behavior. The data is updated every three days. For a more detailed analysis of data related specifically to Airbnb both in New Orleans and in many other cities, please visit Inside Airbnb.

Get the Data

Our data is available for download as simicolon-delimited text files, or through our API. If you use our data, we strongly encourage you to donate to our volunteer-run project in order to help keep our servers running.

Download the delimited text files here: Airbnb | HomeAway

To access the data as geojson through our API, point your application to nolarentalreport.com/rentaltracker and pass the following query parameters:

Parameter Arguments Description
provider air, hma "air" will return all Airbnb listings. "hma" will return all HomeAway listings.
pastweek true Set to true if you would only like to view listings that have been collected in the past week.
neworleans true Set to true if you would only like to retrieve New Orleans listings. Returns listings for Orleans and sorrounding parishes by default.
id e.g. air79609, hma631004vb Unique listing identifier
